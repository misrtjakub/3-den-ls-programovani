Math knihovna

Co je Math?

  • Vestavěná knihovna v C#
  • Pracuje s čísly - např. odmocniny, zaokrouhlování
  • Statická třída - volá se bez vytváření objektu

Math.Abs()

  • Vrací absolutní hodnotu (bez znaménka)
  • Math.Abs(-10) → 10

Ukázka

Math.Round()

  • Zaokrouhluje číslo na celé číslo
  • Math.Round(3.6) → 4

Ukázka

Math.Sqrt()

  • Vrací druhou odmocninu
  • Math.Sqrt(25) → 5

Ukázka

Math.Pow()

  • Umocňuje číslo
  • Math.Pow(2, 3) → 8

Ukázka

Math.Min() a Math.Max()

  • Math.Min(5, 9) → 5
  • Math.Max(5, 9) → 9

Ukázka

Úkoly

  • 1. Zeptej se uživatele na dvě čísla a vypiš jejich maximum
  • 2. Zeptej se na desetinné číslo a zaokrouhli ho
  • 3. Vypočítej 5 na druhou a druhou odmocninu z 81

Metody

Metody

  • Blok kódu, který můžeme opakovaně volat
  • V C# jsou součástí tříd a struktur
  • Zlepšují čitelnost a přehlednost kódu

Anatomie metody

  • static - metoda patří třídě, ne objektu
  • void - návratový typ (žádná návratová hodnota)
  • Název - např. NapisNavod
  • Závorky - seznam argumentů (může být prázdný)
    • static void NapisNavod(string nadpis, string popis)
{
    Console.WriteLine(nadpis);
    Console.WriteLine(popis);
}

Modifikátory metod

  • static - metoda patří třídě, ne objektu
  • public - přístupná odkudkoliv
  • private - přístupná pouze ve stejné třídě
  • protected - přístupná v rámci třídy a dědičnosti
  • internal - přístupná ve stejném projektu

Návratové typy metod

  • void - nevrací nic
  • int - vrací celé číslo
  • string - vrací text
  • bool - vrací pravda/nepravda
  • double, decimal - desetinná čísla

Třídy

Co je třída?

  • Šablona pro vytvoření objektů
  • Popisuje vlastnosti a chování (data a metody)
  • Např. třída Auto může mít vlastnosti barva, rychlost a metodu Jezdi()

Jak vypadá třída?

class Auto {
  public string Barva;
  public int Rychlost;

  public void Jed() {
    Console.WriteLine("Auto jede!");
  }
}

Vytvoření objektu

Auto mojeAuto = new Auto();
mojeAuto.Barva = "červená";
mojeAuto.Jed();

Vlastnosti a metody

  • Vlastnosti - proměnné uvnitř třídy
  • Metody - funkce, které může objekt provádět
  • Každý objekt má vlastní hodnoty těchto vlastností

Konstruktor

class Auto {
  public string Barva;

  public Auto(string barva) {
    Barva = barva;
  }
}

Speciální metoda volaná při vytváření objektu

Úkoly

  • 1. Vytvoř třídu Osoba se jménem a věkem
  • 2. Přidej metodu Pozdrav(), která vypíše „Ahoj, já jsem...“
  • 3. Vytvoř 2 různé objekty typu Osoba a zavolej na nich Pozdrav()

OOP

Programování pomocí objektů

Co je OOP?

  • Programování, kde si věci představujeme jako "objekty"
  • Pomáhá nám psát přehlednější a znovu použitelný kód

Hlavní pojmy

  • Třída - plán, podle kterého se vyrábí objekt (např. recept)
  • Objekt - věc vytvořená podle třídy (např. dort podle receptu)
  • Vlastnosti - co objekt má (např. jméno, barvu, věk)
  • Metody - co objekt umí dělat (např. mňoukat, jet, skákat)

Příklad třídy

class Auto {
  public string Znacka;
  public int Rok;

  public void Start() {
    Console.WriteLine("Motor spuštěn.");
  }
}

Vytvoření objektu

Auto mojeAuto = new Auto();
mojeAuto.Znacka = "Škoda";
mojeAuto.Rok = 2020;
mojeAuto.Start();

Konstruktor

Speciální metoda, která se spustí, když se objekt vytvoří

class Auto {
  public string Znacka;

  public Auto(string znacka) {
    Znacka = znacka;
  }
}

Zapouzdření

Když si objekt chrání svá data

  • Používáme private - skryjeme data
  • A použijeme metody get a set, když je chceme změnit

Ukázka zapouzdření

Dědičnost

  • Jedna třída může vycházet z jiné
  • Získá všechny její vlastnosti a metody
  • Např. class SportovniAuto : Auto

Ukázka dědičnosti

Polymorfismus

  • Objekty se umí chovat různě podle toho, jakou mají třídu
  • Např. Zvire má metodu Zvuk(), ale každé zvíře vydá jiný zvuk
  • Používáme override, když chceme přepsat metodu

Ukázka polymorfismu

Úkoly

  • 1. Vytvoř třídu Zvire s vlastností Jmeno a metodou Zvuk()
  • 2. Vytvoř třídy Kocka a Pes, které dědí ze Zvire
  • 3. V každé třídě přepiš metodu Zvuk()
